Hacer click sobre el numero que sale del bolillero
Para cerrar el cartel tendrá que presionar la tecla C o el botón Volver.

Debajo mostrara los numeros seleccionados en orden.


TECLAS FUNCIONALES Y COMO USARLAS

D = borrar de a un numero 

Al presionar "D" aparecerá el cartel de Modo Borrado. En este momento al clickear un número previamente seleccionado, se deseleccionará. Para cerrar el Modo Borrado debe presionar nuevamente la tecla D.

W = borrar todos los numero

Al presionar "W" todos los números seleccionados previamente en el tablero serán borrados.

Z = salir

Al presionar Z preguntará si desea salir, en este momento presionando otra vez Z se termina el programa, como también seleccionando el botón salir. En caso de apretar el botón volver, mostrará nuevamente el tablero.

L = Linea
Aparece un cartel que dice "Linea", si presiona la tecla C se muestra otra vez el tablero, como también presionando el botón Volver.

B = Bingo
Aparece un cartel que dice "Bingo", si presiona la tecla C se muestra otra vez el tablero, como también presionando el botón Volver.

V=Venta De Cartones
Aparece un cartel que dice "Venta De Cartones", si presiona la tecla C se muestra otra vez el tablero, como también presionando el botón Volver.

O=Muestra el Orden
Aparece un cartel como si clickearamos un numero pero en blanco de esta forma se puede ver que numeros salieron sin agregar un numero que no salio, si presiona la tecla C se muestra otra vez el tablero, como también presionando el botón Volver.
